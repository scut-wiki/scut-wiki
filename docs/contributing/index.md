# 参与共建

欢迎通过 GitHub、Issue、Pull Request 或邮件参与华工手册建设。

## 你可以贡献什么

- 修正过期信息。
- 补充官方来源。
- 添加校区、学院、专业、课程、组织和服务入口。
- 整理一份新生、学习、生活、交通或飞跃经验。
- 把已有内容拆成更清晰的结构化数据。
- 帮忙检查链接、格式、frontmatter 和构建。

## 基本要求

- 尽量标注来源。
- 能区分事实、观点和个人经验。
- 明确信息适用范围：校区、学院、专业、年级、身份和时间。
- 不上传无授权资料。
- 不公开他人隐私、账号、群聊敏感内容或未经同意的个人信息。

## 课程攻略与资料索引

课程攻略结构参考 [SCUT-FT-Guide](https://github.com/OpenFuTech/SCUT-FT-Guide)、浙江大学课程攻略共享计划、清华大学计算机系课程攻略、哈工大深圳课程攻略等开源共享项目。

如果你希望补充课程资料，请尽量按以下结构整理：

- 按年份：例如 `2023`、`2024`、`2025`。
- 按学院或专业：例如未来技术学院、吴贤铭智能工程学院、微电子学院等。
- 按课程类型：基础必修、专业基础、专业选修、通识选修、实践项目。
- 每门课程至少包含：课程名称、适用年级、适用专业、授课学期、贡献者、最后核验时间、资料来源、许可说明。

可以贡献：

- 课程介绍和学习建议。
- 官方课程大纲、培养方案链接。
- 经验帖和复习建议。
- 作业、试卷、课件等资料的索引。

请勿上传不具备分享授权的资料。对于课件、试卷、教材扫描件等内容，优先提交链接和许可说明。

更完整说明见仓库根目录 [CONTRIBUTING.md](https://github.com/scut-wiki/scut-wiki/blob/main/CONTRIBUTING.md)。
