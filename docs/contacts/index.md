## 广州国际校区往返五山校区班车

以下表格采用广州国际校区官网目前可公开检索到的最新完整学生班车通知。乘车前需在 `GZIC 智慧服务` 小程序中的“交通车预约”预约车票；如学校另发专项通知、考试周调整或重大活动加开车次，以当期公告为准。

### 学生班车

| 时段          | 国际校区 -> 五山校区                                 | 五山校区 -> 国际校区                                          |

| 周末 / 节假日 | `08:00`、`13:00`                                     | `13:00`、`17:45`、`20:30`、`22:00`                            |

发车点说明：

- 国际校区：`D5` 北侧道路
