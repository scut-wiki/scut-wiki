## 学生组织和校园自媒体

学生会、SCUT 学生创新俱乐部、华南鲤咕、鲤工包打听、小灯神 GENIE、信息阁、GZIC 小圈等组织和平台都曾参与校园信息流通。它们的性质、更新频率和商业属性可能不同，请自行甄别。

特别提醒：第三方小程序或平台如果要求输入统一认证、教务系统账号密码，应注意信息安全。涉及账号、密码、课表、成绩等敏感信息时，请优先使用学校官方系统。
