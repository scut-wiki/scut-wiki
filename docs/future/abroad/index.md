# 出国深造

## 动机

为什么要出国留学？每个人会有不同答案。无论已经准备留学，还是仍在犹豫，以下问题都值得和家人、师长、朋友收集资料后反复斟酌：

- 我是否要出国留学？
- 我为什么要出国？
- 出国对我意味着什么？
- 我期望通过出国获得什么？
- 一年花费大概是多少？
- 家庭是否有能力支持我完成学业？
- 出国留学让我放弃了哪些机会？
- 我对未来的规划是什么？
- 我是否想继续深造，攻读硕士或博士？

上海交通大学生存手册关于出国有一句提醒：希望你拥有的是一个“积极”的理由，也就是为了得到你想要的某样东西而选择出国，而不是因为厌倦当前生活、想逃避现状。

## 英语考试：IELTS

时间：机考一般除节假日外每天都有数场，纸笔考试通常隔周周六或周日。  
一般目标值：A 类学术类至少 6.0 以上，具体看学校要求。  
模式：机考或纸笔考试。

组成：

- Listening：对话、独白、学术对话和学术独白，题型包括选择、填空、表格、配对、地图等。
- Reading：填空、选择、判断、信息题、配对、段落大意、流程图和图表题等。
- Writing：小作文描述和分析数据，大作文讨论问题并展开论证。
- Speaking：日常对话、话题论述和话题讨论。

## 英语考试：TOEFL

模式：机考。

组成：

- Reading：选择题，包括细节、推理、单词、句子插入、段落大意等。
- Listening：对话和学术讲座，题型包括单选、多选和表格题。
- Speaking：独立题、阅读听力综合题、概念解释题和学术听力复述题。
- Writing：综合写作和学术写作。

## 其他语言考试

- Duolingo：待补充。
