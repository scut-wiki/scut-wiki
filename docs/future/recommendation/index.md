# 保研与直博

## 夏令营

时间通常集中在 3-8 月。夏令营是相对预推免难度更大的保研途径，考核形式包括但不限于笔试、极限作业、面试等。优点是早上岸、早联系导师；缺点是准备压力更大，不同学校和学院差异明显。

## 预推免

时间通常集中在 8-9 月。预推免难度相对夏令营小一些，考核形式多为面试。由于夏令营阶段会有大量 offer 流动，预推免仍有机会，但可选导师和方向可能减少。

## 常见材料

以下材料请根据目标院校和学院要求准备，具体以当年通知为准：

- 个人简历。
- 个人陈述。
- 学院成绩单和排名证明原件及彩色扫描件。
- 身份证正反面彩色扫描件。
- 学生证封面和里页彩色扫描件。
- 四六级、GRE、GMAT、IELTS、TOEFL 等英文水平证明。
- 竞赛获奖证书。
- 奖学金证书。
- 荣誉称号证书。
- 已发表论文证明材料。
- 专利证明材料。

有的文件建议准备好单件 PDF。

## 个人简历

可包含：

1. 学术背景：学校、专业、GPA、排名、英文水平等。
2. 研究经历：经历从近到远，说明自己承担的工作、问题、方法、算法或理论、结果。
3. 荣誉奖项：突出奖项名称，相同奖项可合并年份。
4. 社团经历：非必须，可体现课内外能力。
5. 专业技能：语言、工具、证书和相关知识。

## 个人陈述

很多学校的特殊表格和文书都是基于简历和个人陈述延伸。个人陈述应说明自己的背景、兴趣、经历、选择该方向的原因、未来计划和与目标项目的匹配。

## 阅读建议

保研和直博经验强依赖学院政策、专业方向、当年竞争和个人背景。不要只看“上岸结果”，更要看材料准备、时间线、选择逻辑和失败教训。
