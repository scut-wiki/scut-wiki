# 就业

## 就业

就业经验会受到行业周期、城市、岗位、学历、实习经历和个人能力影响。阅读时应重点关注方法和时间线，而不是只看最终去向。

可关注：

- 不同专业常见行业和岗位。
- 简历、笔试、面试、实习准备。
- 校招时间线。
- 不同城市和行业选择。
- 技术岗、产品岗、运营岗、设计岗、管培、国企、事业单位等路径差异。
