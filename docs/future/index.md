# 华南理工大学飞跃手册

飞跃手册关注保研、出国深造、考研、就业、考公考编或其他毕业去向经验。力求为 “如何在华南理工大学为申请国内、海外高校和入职优秀企业做出更全面的准备“ 这一问题作出解答。

## 阅读提醒


由于外界经济形势、就业形势、热门专业、招生政策都会变化，每位同学的个人情况也不相同，阅读往年飞跃经验时应特别注意时效性和契合度：

- 作者背景与自己的差异。
- 经验对应的年份、学院、专业和申请方向。
- 政策、考试和项目要求的最新官方版本。
- 经验观点与事实信息的区别。

重要的不是照搬某个成功案例，而是理解前人如何搜集信息、评估选择、准备材料、修正路径，并把这些方法转化为自己的行动。
