# 吴贤铭智能工程学院

## 机器人工程

机器人工程会涉及机械、电子、计算机、自动化等方向，课程内容较杂。这个专业更像是在帮助学生找到一两个感兴趣方向，然后自己深入学习。

如果对 Robotics 感兴趣，可以关注机器人领域前沿团队和个体，例如 Boston Dynamics、Tesla Bot、稚晖君等，也可以参加 RoboMaster、Robocon 等机器人比赛。华工五山校区有华南虎战队，国际校区有 RobotIC 机器人实验室。

推荐阅读：[机器人工程师学习计划](https://zhuanlan.zhihu.com/p/22266788)。

## 智能制造

智能制造专业融合智能技术、机械工程、电子工程和控制工程。相关课程包括工程导论、Python、数据结构、工业大数据、工程力学、电路导论、热力学、计算流体力学、动力学与振动导论、设计与制造、人工智能、金工实习等。

其中工程导论和设计制造类实践课会要求学生组队完成工程项目，需要自学建模、单片机、展示、分工和协作。
