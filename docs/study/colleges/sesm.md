# 前沿软物质学院

前沿软物质学院依托华南软物质科学技术研究院建立，专业方向围绕分子设计与合成、多尺度结构调控、软物质功能和智能先进杂化材料、极端条件下特种材料、信息与能源先进材料、健康及医学先进材料等。

该方向可能适合对化学、物理和材料感兴趣的同学。师资多为年轻海归博士，也可在学院官网查看老师研究方向。
