# 生物医学工程学院

生物医学工程是交叉学科，分支方向包括生物材料、生物电子、医疗器械等。该专业优势在于开放的研究氛围：院长和老师鼓励本科生参与科研，各实验室对学生长期开放，可联系老师申请参观学习并加入感兴趣课题组。

需要注意的是，交叉学科往往广而不精。本科阶段课程较全面，真正深入某一方向通常还需要自学和科研训练。国内相关产业对本科生直接就业需求可能有限，如想做研究，读研读博会更常见。
