# 微电子和集成电路学院

国际校区本科生的新工科专业采用全英 / 双语教学，官方介绍可参考：[广州国际校区本科生专业介绍](http://www2.scut.edu.cn/gzic/30309/list.htm)。

## 集成电路与微电子

微电子和集成电路都服务于芯片设计和制造。一般可以理解为集成电路更偏设计，微电子更偏制造，但本科课程差异不一定很大。

这个方向本科就业相对少见，很多学生会继续读硕士或博士。核心基础包括数字电路、模拟电路、信号与系统、电磁场、数字集成电路设计、模拟集成电路设计等。

## 学习共性

几个共性建议：

- 越早选择方向，越早积累能量。
- 选择自己真正想做的事情。
- 在合适的地方做合适的事情。
- 根据考研、保研、出国、技术路线、体制路线等不同目标分配时间。

对于数学学习，可以分为以成绩为导向、以会用为导向，或两者兼顾。以成绩为导向需要熟悉题型和考试要求；以会用为导向需要理解推导和原理，并使用 MATLAB、Python 等工具解决实际问题。
