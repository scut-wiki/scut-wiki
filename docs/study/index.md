# 学习篇

学习篇收集教务系统、学生手册、培养方案、课程信息、学习建议、学院与专业经验。

## 学院与专业经验

我们收集了一些专业学习的经验分享，已整理到[学院与专业经验](/study/colleges/)。

## 教务系统

对于学生来说，教务系统是非常重要的网址，可以查成绩、课程安排、已修学分等信息。

### 直接访问

[华南理工大学教学管理信息平台](http://jw2018.jw.scut.edu.cn/)

### 通过统一门户访问

点击[华南理工大学官网](https://www.scut.edu.cn/new/)右上角“门户”进入[统一门户](https://my.scut.edu.cn/up/)，再从“教务服务”进入“教务系统（学生端）”。

![统一门户](/legacy/assets/华南理工大学统一门户20230818234043.png)

## 学生手册

每年的本科生学生手册可以从 [教务处 / 管理制度 / 学生学习 / 学生手册](https://jw.scut.edu.cn/zhinan/cms/category/index.do?id=ff808081696f96b101696fd755450000) 获取。

每年的研究生学生手册可以从 [研究生院 / 综合信息 / 研究生手册](https://www2.scut.edu.cn/graduate/yjssc/list.htm) 获取。

早期归档中保存过 2021、2022 年本科生和研究生手册 PDF。阅读学生手册时优先使用教务处和研究生院的最新入口。

## 本科生培养方案

培养方案 / 培养计划可以从各学院官网的本科生培养计划中获取，例如：

- [生物医学科学与工程学院 / 招生培养 / 本科生培养 / 专业介绍](https://www2.scut.edu.cn/bmse/zyjs/list.htm)
- [吴贤铭智能工程学院 / 人才培养 / 本科生教育 / 培养计划](http://www2.scut.edu.cn/wusie/34049/list.htm)
- [前沿软物质学院 / 本科生教育 / 招生专业](https://www2.scut.edu.cn/SESM/_t3640/33029/list.psp)
- [微电子学院 / 招生培养 / 本科生培养 / 培养计划](https://www2.scut.edu.cn/microelectronics/pyjh/list.htm)
- [未来技术学院 / 招生与培养 / 本科生招生与培养 / 培养计划](https://www2.scut.edu.cn/ft/pyjh/list.htm)

本专业培养计划也常见于学院教务 QQ 群或学院内部通知，且可能隔一段时间更新一次。请以最新官方文件为准。

## 课程简介和教学大纲

华工暂未见到公开可查所有课程简介和教学大纲的平台。部分基础课可以从授课学院官网查询，例如数学学院课程资源：[http://www2.scut.edu.cn/math/10414/list.htm](http://www2.scut.edu.cn/math/10414/list.htm)。

有些学生组织和校园媒体会制作选课指南，例如：

- 鲤工包打听：[选课推荐汇总 | 快来 get 最适合你的课程！](https://mp.weixin.qq.com/s/hb2NRpsMr7fWFQKkfjSamA)
- 华南鲤咕：[干货 | 你想要的『这么棒的走心』通选课指南](https://mp.weixin.qq.com/s/-lZc5fORLvPnxMQMtuUFQw)

这些内容有较强时效性，阅读时应注意发布时间、适用校区和课程政策变化。

## 学习建议与攻略

### 认识你的专业

可以根据学生手册、培养计划、课程简介、教学大纲、学长学姐经验、教师教授建议、行业从业者观点和互联网公开资料认识专业。

可参考华南鲤咕制作的专业介绍推文：[华工最真实的专业介绍来啦，选专业还在犹豫什么！](https://mp.weixin.qq.com/s/GKDQaXg-9Was6U1AP-Mpkw)。

### 转专业

可参考：[微协攻略 | 华工【转专业】如何转？最全转专业攻略请查收！](https://mp.weixin.qq.com/s/9H4tEz_en_RlEzUvvgHbDw)。

转专业政策每年可能变动，请以当年官方文件为准。

### 学生邮箱

学生邮箱很重要。使用 edu 后缀邮箱给他人发邮件时能证明学生身份，很多互联网和软件服务也可以通过学生认证 / 教育优惠享有便利。

如果在门户里看到邮箱还未注册，可进入 [https://stuemail.scut.edu.cn](https://stuemail.scut.edu.cn) 填写信息注册。

![学生邮箱入口示意](</legacy/assets/image (3).png>)

### 提高信息检索能力

建议掌握一定搜索技巧：

- 用关键词搜索，避免只用自然语言描述。
- 使用 `site:example.com` 限定站点。
- 使用 `filetype:pdf`、`filetype:docx` 限定文件类型。
- 注意信息源：一手信息还是转载加工，事实还是观点，适用范围和发表时间。

AI / GPT 等工具可以帮助生活、学习和信息整理，但应保留来源意识和独立判断。建议继续阅读[信息素养](/literacy/)。

## 信息检索能力

很多重要事项每年在相近时间段发布，通知通常按时间排序。把相关网站过去一年的通知大致浏览一遍，有助于理解推免、综测、奖学金、评优、竞赛和实践项目的节奏。

建议配合阅读：[信息素养](/literacy/)。

以下是一些官方通知渠道：

| 事项                                             | 渠道                                          |
| ------------------------------------------------ | --------------------------------------------- |
| 保研推免、综测、奖学金、评优评先、互联网+ 大赛等 | 华南理工大学官网 - 门户 - 信息中心 - 公务通知 |
| SRP、大创、百步梯攀登计划、互联网+ 等实践信息    | 教务处官网 - 通知公告 - 实践                  |
| 挑战杯、各类奖学金等                             | 共青团华南理工大学委员会官网 - 通知公告       |
| 学院比赛、评优评先、公示、活动                   | 各学院官网 - 通知公告                         |
