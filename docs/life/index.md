## 网络

- 教学区、办公区和图书馆等公共区域的校园无线网络名称通常为 `scut`。
- 宿舍无线网络名称通常为 `scut-student`。本科生宿舍 Wi-Fi 在工作日过凌晨0点后会断网。
- 登录方式均以学校统一认证账号为主。

如果设备连上 Wi-Fi 但没有自动跳出认证页：

- 直接访问 `s.scut.edu.cn`
- 或访问 `http://captive.apple.com` 触发认证页

来源：

- [网络与信息化办公室](https://web.scut.edu.cn/15718/list.htm)

## 国际校区门禁

- 校门、人行闸机和部分楼栋已支持校园卡和人脸识别。
- 人脸采集与授权路径：微信 -> 通讯录 -> 华南理工大学 -> `Mypass 我的通行证`。
- 需要同时完成“采集”和“校园应用授权”，同步到闸机通常要等待约 `24` 小时。
- 校区一站式物业服务站也可协助处理门禁与现场求助。

来源：

- [华南理工大学人脸特征照片采集指南](https://www2.scut.edu.cn/gzic/2024/1008/c30728a563842/page.htm)

### 三校区与广州公共交通

#### 五山校区

- 地铁：`3` 号线五山站最常用；去天河方向最方便。
- 公交线路调整较频繁，建议直接在 `广州交通·行讯通` 查询“华工大”“华工大总站”。

#### 大学城校区

- 地铁：`4` 号线、`7` 号线大学城南站。
- 前往国际校区的校外接驳，学校官网目前主要推荐 `番87` 和地铁换乘。

#### 广州国际校区

校区官网当前公开的主要公交与换乘信息如下：

- `番10`、`番17`、`番186` 到华工国际校区公交总站
- `番72`、`番87` 到校区北门方向
- `番87` 可在大学城和新造地铁站之间接驳国际校区

`番87` 校园段最新公开说明：

- `D5/华工北门 -> 大学城体育中心` 常规线一般逢 `30` 分发班
- 去地铁新造站的短线一般逢整点发班
- 实时位置可用 `广州交通·行讯通` 查询

## 社团与生活信息的使用方式

学生组织、社团、校园媒体和书院公众号仍然是获取一手生活信息的重要渠道，但它们的更新频率和侧重点不同。查生活问题时，建议顺序是：

1. 学校 / 校区官网 / 师生服务中心、网络与信息化办公室、书院或学院等官方渠道
2. 微信公众号自媒体（例如华南哩咕、鲤工包打听等）也会产出很多生活攻略
3. 同学经验

这样最不容易被过期截图和口口相传误导。
