# 开放资料索引

华工手册会把常用入口、校区资源、学生组织、友链和参考项目等资料整理成可复用的开放索引。这样做是为了让页面、搜索和后续工具都能使用同一份可靠资料，减少重复整理。

## 当前收录方向

- 校区与校园服务入口。
- 官方网站、系统入口和常用链接。
- 学生组织、社团和校园媒体。
- 友链、参考项目和开放共享项目。

## 使用提醒

索引内容会随学校部门、系统入口和组织状态变化而变化。使用时请优先核对官方来源、发布时间和适用校区；发现过期信息时，可以在 GitHub 或通过邮件 `scut.wiki@outlook.com` 反馈。
